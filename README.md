# tsw_project
Konie, sędziowie i czarownica


### Instalacja zalezności
```
$ cd data_provider
$ npm install
```
```
$ cd nodejs_api
$ npm install
```
```
$ cd vue
$ npm install
```

### Seedowanie bazy
```
$ cd data_provider
$ npm start
```
```
$ cd nodejs_api
$ npm run seed-data
$ npm run seed-user
```


### Uruchamianie
W dwóch oknach/kartach terminala:
```
$ cd nodejs_api
$ npm start
```
```
$ cd vue
$ npm start
```
